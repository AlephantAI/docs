This page helps you get started with Alephant AI Gateway [chat models](/oss/langchain/models). The `langchain-alephantai` package provides `ChatAlephantAI`, a chat model configured to send requests through Alephant AI Gateway with Alephant session headers.

## Setup

To access models through Alephant AI Gateway, create an Alephant Virtual Key and install the `langchain-alephantai` integration package.

### Installation

```bash
pip install -U langchain-alephantai
```

### Credentials

Set your Alephant Virtual Key as the `ALEPHANT_API_KEY` environment variable:

```python
import getpass
import os

if "ALEPHANT_API_KEY" not in os.environ:
    os.environ["ALEPHANT_API_KEY"] = getpass.getpass("Enter your Alephant API key: ")
```

To enable automated tracing of your model calls, set your [LangSmith](/langsmith/home) API key:

```python
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")
os.environ["LANGSMITH_TRACING"] = "true"
```

## Instantiation

Now you can instantiate the chat model:

```python
from langchain_alephantai import ChatAlephantAI

llm = ChatAlephantAI(
    api_key=os.environ["ALEPHANT_API_KEY"],
    model="gpt-4o-mini",
    session_name="langchain-chat",
)
```

`ChatAlephantAI` also accepts keyword arguments supported by `langchain_openai.ChatOpenAI`, such as `temperature`, `timeout`, and `max_retries`.

```python
llm = ChatAlephantAI(
    api_key=os.environ["ALEPHANT_API_KEY"],
    model="gpt-4o-mini",
    session_name="langchain-chat",
    temperature=0,
    timeout=30,
    max_retries=2,
)
```

## Invocation

```python
messages = [
    (
        "system",
        "You are a helpful assistant that translates English to French. Translate the user sentence.",
    ),
    ("human", "I love programming."),
]
ai_msg = llm.invoke(messages)
print(ai_msg.content)
```

## Session metadata

Alephant session metadata is attached through gateway request headers. Pass `session_name` or `session_id` directly, or pass an `AlephantGatewayContext`.

```python
from alephantai import AlephantGatewayContext
from langchain_alephantai import ChatAlephantAI

context = AlephantGatewayContext(session_name="customer-support-eval")

llm = ChatAlephantAI(
    api_key=os.environ["ALEPHANT_API_KEY"],
    model="gpt-4o-mini",
    context=context,
)
```

## Custom gateway URL

Use `base_url` when testing against another Alephant AI Gateway endpoint.

```python
from langchain_alephantai import ChatAlephantAI

llm = ChatAlephantAI(
    api_key=os.environ["ALEPHANT_API_KEY"],
    model="gpt-4o-mini",
    session_name="langchain-chat",
    base_url="https://ai.alephant.io/v1",
)
```
